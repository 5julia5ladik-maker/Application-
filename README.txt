HomeStock Universal

Что изменено:
- один сервер отдает и frontend, и backend
- приложение открывается одинаково на ПК и телефоне
- телефон подключается по ссылке вида http://IP_ПК:8000
- адрес для телефона показывается прямо на главном экране приложения
- Gemini API key берется из переменной окружения GEMINI_API_KEY
- данные товаров теперь общие для ПК и телефона, потому что хранятся на сервере

Запуск на ПК:
1. Открой папку проекта
2. Открой терминал в этой папке
3. Выполни:
   set GEMINI_API_KEY=ТВОЙ_GEMINI_API_KEY
4. Запусти:
   start_server.bat
5. Открой в браузере:
   http://127.0.0.1:8000

Запуск на телефоне:
1. Подключи телефон и ПК к одной Wi‑Fi сети
2. Узнай IP компьютера командой:
   ipconfig
3. Найди IPv4 Address, например 192.168.0.147
4. На телефоне открой:
   http://192.168.0.147:8000

Проверка:
- http://127.0.0.1:8000/health на ПК
- http://IP_ПК:8000/health на телефоне

Если телефон не открывает страницу:
- разреши Python / uvicorn в брандмауэре Windows
- проверь, что сервер запущен
- проверь, что устройства в одной сети
